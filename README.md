# Learnico – Online Courses Website

A responsive web application for an online courses platform built with React.

## Live Demo

https://fabulous-strudel-b6b089.netlify.app/

## GitHub Repository

https://github.com/AndriyLebovich/Learnico__react

## Features

- Responsive user interface for desktop and mobile
- Reusable React components
- Interactive UI elements (slider, navigation menu, modal window)
- Clean and modern UI design

## Technologies

- React
- JavaScript
- HTML5
- CSS3
- Responsive Design
